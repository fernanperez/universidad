Comandos basicos de GIT

  • git config --global user.name "nombre-de-usuario": Con este comando agregamos el nombre del desarrollador. Ejemplo: git config --global user.name "fulanito perez"
  • git config --global user.email "email-de-usuario": Con este comando agregamos el email del desarrollador. Ejemplo: git config --global user.email "fulanito.perez@example.com"
  • git init: para inicializar el git.
  • git add .: Comando para guardar todos los archivos modificados.
  • git add archivo_modificado: Comando para guardar un archivo modificado especifico.
  • git branch: Comando que muestra todas las ramas que hay, y con asterico la rama seleccionada.
  • git branch -M nombre_rama: Comando para renombrar la rama. Ejemplo: gir branch -M rama_2
  • git branch nombre_de_rama: Comando que crea una nueva rama, pero no la selecciona. Ejemplo: git branch rama_2
  • git checkout nombre_de_rama: Comando que selecciona una rama. Ejemplo: git checkout rama_2
